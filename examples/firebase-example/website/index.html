Firesearch blog demo

This JavaScript/HTML app is a simple, searchable blog.

How does it work?

The front-end puts blog posts in Firestore via the Firebase SDK.

Cloud Functions running in Google Cloud Platform respond to the documents and automatically add it to a Firesearch index.

This front-end app then performs searches with the Firesearch JavaScript client.

What would you like to do?

  • View blog posts
  • Create a new blog post
  • Learn how to deploy this app yourself
